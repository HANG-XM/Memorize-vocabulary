
# 智能背单词

一个基于 PyQt6 开发的智能背单词应用，支持多种学习模式和单词本管理功能。

## 项目概述

智能背单词是一个功能丰富的桌面应用程序，旨在帮助用户高效学习和记忆英语单词。应用提供多种学习模式（认识/不认识、选择释义、拼写单词），支持自定义单词本管理，并具有友好的用户界面。

## 主要特性

- 📚 **单词本管理**：创建、删除、编辑单词本
- ➕ **单词管理**：添加、修改、删除单词
- 🎯 **多种学习模式**：
  - 认识/不认识模式
  - 选择释义模式
  - 拼写单词模式
- 💾 **本地数据库存储**：使用 SQLite 保存学习数据
- 🎨 **现代化 UI**：采用 PyQt6 构建美观界面，包含动画效果

## 前置条件

- Python 3.8 或更高版本
- PyQt6 库

## 安装依赖

```bash
pip install PyQt6
```

## 运行方法

1. 克隆或下载项目代码
2. 进入项目目录：
   ```bash
   cd src
   ```
3. 运行主程序：
   ```bash
   python main.py
   ```

## 使用说明

1. **启动应用**：运行 `main.py` 启动程序
2. **单词本管理**：
   - 点击"单词本管理"进入管理界面
   - 创建新单词本或管理现有单词本
3. **添加单词**：
   - 选择单词本后添加新单词及其释义
4. **学习设置**：
   - 选择要学习的单词本和学习模式
5. **开始学习**：
   - 根据设置的模式进行单词学习

## 项目结构

```
src/
```
